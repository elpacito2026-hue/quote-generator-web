# 🌐 Quote Generator Web

A simple and interactive web app that generates random motivational quotes with a single click.

---

## ✨ Features
- 🎯 One-click quote generation
- 💬 Clean and minimal interface
- ⚡ Instant response (no reload needed)

---

## 🚀 How to Use
1. Open `index.html` in your browser  
2. Click on the **"Get Quote"** button  
3. Enjoy a new quote every time  

---

## 🛠️ Tech Used
- HTML  
- JavaScript  

---

Built to practice basic web development and create a simple interactive user experience.
